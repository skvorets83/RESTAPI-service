# RESTAPI-service
Система авто индексации python - кода. Позволяет найти функцию или класс и понять, что она делает.

Проблема:
В проекте 30 файлов, документации нет.
Новые разработчики тратят много времени на поиск нужного фрагмента кода.

Решение: Автоматическая индексация с возможностью поиска.


''''bash
pip install -r requirements.txt
'''
